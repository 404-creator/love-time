# 恋恋时光 PWA

这是一个不需要服务器、不需要审核的手机网页 App。

## 最终形态aaa

手机浏览器打开网页后，可以添加到桌面，之后像 App 一样使用。

核心特点：

- 数据只保存在当前手机浏览器本地。
- 不依赖微信小程序、云开发、数据库或服务器后端。
- 支持离线打开。
- 支持本地纪念册、本地照片、本地胶囊。
- 支持到期后才能打开胶囊。
- 支持隐私锁。
- 支持同步码、二维码链接、完整备份文件。
- 支持今日心情、纪念日倒计时、恋爱小任务、小惊喜和彩蛋。

## 本地预览

在当前目录启动一个静态服务器：

```bash
npx serve .
```

或者：

```bash
python -m http.server 5173
```

然后访问：

```text
http://localhost:5173
```

PWA 安装和离线缓存更建议用 HTTPS 部署后测试。

## 免费部署

推荐用 Cloudflare Pages、GitHub Pages、Netlify 或 Vercel。它们都能托管纯静态网页。

只需要部署这些文件：

```text
index.html
styles.css
app-pwa.js
manifest.webmanifest
service-worker.js
icons/
vendor/
```

旧的小程序目录和云函数目录可以不用部署。

## 手机安装

iPhone：

1. 用 Safari 打开部署后的链接。
2. 点分享按钮。
3. 选择“添加到主屏幕”。
4. 桌面会出现“恋恋时光”图标。

Android：

1. 用 Chrome 打开部署后的链接。
2. 菜单里选择“添加到主屏幕”或“安装应用”。
3. 桌面会出现“恋恋时光”图标。

## 手动同步

A 手机：

1. 打开“同步”。
2. 点“生成同步码”。
3. 复制同步码发给 B，或者让 B 扫二维码。

B 手机：

1. 如果收到同步码，打开“同步”，粘贴后点“导入”。
2. 如果扫二维码，会自动打开网页并导入。

注意：二维码适合文字数据。照片比较大，建议用“下载备份”和“导入备份”。

## 隐私提醒

隐私锁只保护当前浏览器里的入口，不是端到端加密。因为这是完全本地版，没有服务器保存你的内容。
